# PulseShift PWA

PulseShift is a mobile-first healthcare shift marketplace prototype.

This GitHub Pages build runs without Expo. It uses browser localStorage for demo accounts and shifts, so it opens directly in iPhone Safari and can be added to the Home Screen.

## Use

1. Open the GitHub Pages URL.
2. Choose Healthcare Pro or Facility.
3. Create a demo profile.
4. Claim or post shifts.

Firebase-backed auth, Firestore, and push notifications can be connected after the PWA shell is stable.
